# Sahyog Dairy Cooperative — Frontend Template

A static, responsive frontend template inspired by the layout/structure of a typical dairy cooperative website (hero slider, feature cards, chairman's message, product grid, awards/plant/management carousels, ISO certifications, stats counter, footer).

All text and images in this template are **placeholders** — swap in your own copy, photos, and logo before going live. No copyrighted content, trademarks, or images from any existing organization are included.

## Structure

```
dairy-coop-site/
├── index.html
├── css/style.css
├── js/script.js
├── images/        (empty — add your own assets here)
├── vercel.json
└── README.md
```

## Run locally

Just open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Deploy to Vercel

1. Push this folder to a new GitHub repository.
2. Go to [vercel.com/new](https://vercel.com/new), import the repo.
3. Framework preset: **Other** (static site) — no build command needed.
4. Deploy.

Or via CLI:

```bash
npm i -g vercel
vercel
```
